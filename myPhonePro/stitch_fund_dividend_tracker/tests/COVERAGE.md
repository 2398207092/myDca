### 已捕获的 bug 模式

通过本次测试，已经能检测以下类型的 bug：

1. **成本扭曲**：编辑 costPerShare 后未同步 cost → I2 + A1 捕获
2. **净投入漂移**：交易流水与 netInvestment 不一致 → I6 捕获
3. **市值不同步**：操作后 marketValue 未更新 → I1 捕获
4. **息率计算错误**：dividendRate 用旧 predictedDividend → I5 捕获
5. **数据陈旧**：净值超过 7 天未刷新 → A5 捕获
6. **零份额遗留成本**：清仓后 cost 未归零 → A7 捕获
7. **定投扣款与持仓不一致**：扣了钱但份额没加/加错 → DC1 + I6 + I8 捕获
8. **定投失败未回滚**：净值获取失败或买入失败后现金未恢复 → DC2 捕获
9. **删除交易现金未回滚**：删买入没加回现金 → C6 捕获
10. **余额不足仍扣款**：现金不够但定投照样执行 → DC3 捕获
11. **分红到账未更新持仓**：markDistributed 后 totalDividendReceived 没变 → E1 捕获
12. **分红到账未加现金**：markDistributed 后现金余额没增加 → E2 捕获
13. **已到账事件被取消**：distributed 事件被错误取消 → E4 捕获
14. **总资产汇总错误**：多分类加总不对 → O1 捕获
15. **百分比计算溢出**：分类占比之和不等于 100% → O3 捕获
16. **快照重复创建**：同一天多次快照 → O5 捕获
